# AI点读后台管理系统 — 需求说明文档

## 一、项目概述

### 1.1 项目名称
AI点读 · 后台管理系统

### 1.2 项目目标
为"AI点读"产品提供后台内容管理能力，支持管理员对课本页面进行热区（可点读区域）规划、文字识别、音频生成与管理，最终输出可供前端学生端使用的点读数据。

### 1.3 技术方案
- 纯前端实现（HTML + CSS + JavaScript）
- 数据持久化：localStorage
- 无需后端服务，可直接通过浏览器访问
- 支持 GitHub Pages 静态部署

---

## 二、系统架构

### 2.1 页面布局
系统采用三栏式布局：
- **左侧边栏**：教材选择与章节导航
- **中间主区域**：课本页面展示与热区编辑画布
- **右侧面板**：热区详情编辑与音频管理

### 2.2 文件结构
```
Backstage/
├── index.html              # 主页面
├── css/
│   └── backstage.css       # 样式文件
├── js/
│   ├── app.js              # 主应用控制器
│   ├── data.js             # 数据管理模块
│   ├── hotzone-editor.js   # 热区编辑器模块
│   └── audio-manager.js    # 音频管理模块
├── insidebook/             # 英语课本页面图片
└── insidebookCN/           # 语文课本页面图片
```

## 三、功能需求

### 3.1 教材选择（左侧边栏）

#### 3.1.1 三级联动选择器
系统提供三级下拉选择器，用于定位一本具体的课本：

| 层级 | 名称 | 说明 |
|------|------|------|
| 第一级 | 学科 | 语文、英语 |
| 第二级 | 教材（出版社） | 语文→人教版；英语→沪教版 |
| 第三级 | 课本 | 如：三年级上册、三年级下册、四年级上册(2025秋版) 等 |

- 选择学科后，教材下拉框自动加载对应出版社列表
- 选择教材后，课本下拉框自动加载对应课本列表
- 未选择上级时，下级下拉框禁用并显示提示文字

#### 3.1.2 章节导航列表
选择课本后，下方显示该课本的章节目录：

- **语文**：按单元分组，支持折叠/展开（▾/▸ 切换）
  - 示例：第一单元 · 阅读 → 1 小蝌蚪找妈妈、2 我是什么 ...
- **英语**：按 Unit 分组，支持折叠/展开
  - 示例：Unit 1 How do we feel? → Get ready、Explore ...
- 每个章节前有状态圆点：
  - 🟢 绿色：已完成（所有页面都有热区数据）
  - 🟡 黄色：部分完成
  - ⚪ 灰色：未开始
- 章节列表区域自动填充剩余侧边栏空间，支持滚动

---

### 3.2 课本页面展示（中间主区域）

#### 3.2.1 空状态
未选择章节时，显示空状态提示："请选择教材章节"。

#### 3.2.2 页面图片展示
选择章节后，加载对应课本页面图片：
- 英语课本图片来源：`insidebook/` 目录
- 语文课本图片来源：`insidebookCN/` 目录
- 图片自适应显示，保持比例，不超出可视区域
- 图片加载失败时显示 SVG 占位图（含页码和课本信息）

#### 3.2.3 翻页功能
页面底部居中显示翻页栏：
- 上一页按钮（‹）、页码信息（当前页/总页数）、下一页按钮（›）
- 首页时上一页按钮禁用，末页时下一页按钮禁用
- 按钮与页码间距适中，页码不换行显示

#### 3.2.4 底部状态栏
显示当前操作状态和热区数量统计。

---

### 3.3 工具栏

工具栏位于主区域顶部，分为左右两组：

#### 左侧工具组
| 按钮 | 功能 |
|------|------|
| 🖱️ 选择 | 切换为选择模式，可点击/拖拽热区 |
| ➕ 添加热区 | 切换为绘制模式，在图片上拖拽绘制矩形热区 |
| 🤖 自动识别热区 | 自动在页面上生成预设热区模板 |

#### 右侧工具组（醒目主色调按钮）
| 按钮 | 功能 |
|------|------|
| 开始识别文字 | 批量对所有热区执行 OCR 文字识别 |
| 生成音频 | 打开音频生成弹窗，批量生成音频 |
| 保存全部 | 保存当前页面所有热区数据到 localStorage |

---

### 3.4 热区编辑器

#### 3.4.1 热区绘制
- 在"添加热区"模式下，鼠标在图片上拖拽绘制矩形区域
- 绘制过程中显示蓝色虚线临时矩形
- 松开鼠标后创建热区（最小尺寸 > 1%）
- 新建热区自动编号递增

#### 3.4.2 热区显示
- 未选中热区：灰色虚线边框（无内容）或绿色实线边框（有内容）
- 选中热区：蓝色实线边框，半透明蓝色填充
- 每个热区左上角显示编号标签（如 #1、#2）
- 有音频的热区右上角显示 🔊 图标

#### 3.4.3 热区选择与交互
选中热区后显示：
- **删除按钮**：红色圆形 × 按钮，位于热区右上角，点击删除该热区
- **8个拉伸手柄**：四角 + 四边中点，白色方块蓝色边框，可拖拽调整热区大小
- **拖拽移动**：点击热区矩形本身可拖拽移动位置
- 所有位置使用百分比坐标，适配不同分辨率

#### 3.4.4 自动识别热区
点击"自动识别热区"按钮后：
- 系统自动在页面上生成 9 个预设热区模板
- 热区类型包括：sentence、text、word、paragraph
- 位置带有随机偏移，模拟真实识别效果

---

### 3.5 热区详情面板（右侧）

#### 3.5.1 无选中状态
显示提示文字："点击画布上的热区查看详情"

#### 3.5.2 热区编辑表单
选中热区后显示以下信息：

| 字段 | 说明 |
|------|------|
| 编号 | 只读禁用，灰色背景，显示如 #1 |
| 位置信息 | X、Y、宽、高，四列网格布局，可手动输入调整 |
| 文字内容 | 多行文本框（6行），默认只读禁用 |

#### 3.5.3 文字内容编辑
- 默认状态：只读、灰色背景、不可编辑
- 点击"✏️ 编辑"按钮：切换为可编辑状态，白色背景，光标聚焦
- 编辑完成点击"💾 保存"按钮：
  - 保存文字内容到热区数据
  - 自动保存到 localStorage
  - 自动触发该热区所有音色的音频重新生成
  - 重新生成过程中，音频列表显示"生成中..."蓝色脉冲动画
  - 生成完成后恢复"已生成"状态

#### 3.5.4 音频管理
显示当前热区的音频状态列表，每种音色一行：
- 音色名称（标准男声、甜美女声、可爱童声）
- 状态标签：已生成（绿色）/ 待生成（黄色）/ 失败（红色）/ 生成中...（蓝色脉冲）
- 试听按钮（▶）：仅"已生成"状态可用，使用 Web Speech API 模拟播放

#### 3.5.5 全部热区列表
- 横向排布，紧凑显示
- 每个热区项包含：编号圆圈 + 状态图标（🔊/📝/⬜）
- 点击可快速切换选中热区
- 最大高度 120px，超出滚动

---

### 3.6 音频生成

#### 3.6.1 音频生成弹窗
点击工具栏"生成音频"按钮打开弹窗：

**音色选择区域**：
- 标准男声 ☑️
- 甜美女声 ☑️
- 可爱童声 ☑️
- 默认全部勾选，至少选择一种

**进度显示区域**（生成开始后显示）：
- 当前进度文字：热区编号 + 音色名称
- 百分比数字
- 进度条（蓝色填充动画）
- 完成数/总数

**按钮**：
- 取消：停止生成并关闭弹窗
- 开始生成：启动批量生成
- 生成完成后按钮变为"确定"，点击关闭弹窗

#### 3.6.2 音频生成逻辑
- 模拟 TTS 生成，每个音频 0.5~2 秒随机延迟
- 5% 概率模拟生成失败
- 生成完成后自动保存音频状态到热区数据
- 支持取消操作

#### 3.6.3 音频播放
- 使用浏览器 Web Speech API 模拟播放
- 根据音色调整 pitch 和 rate：
  - 标准男声：pitch 0.8
  - 甜美女声：pitch 1.3
  - 可爱童声：pitch 1.6, rate 1.0
- 自动识别中英文（根据是否包含中文字符）

---

### 3.7 批量文字识别（OCR）

点击工具栏"开始识别文字"按钮：
- 遍历当前页面所有热区
- 每个热区间隔 300ms 执行模拟 OCR
- 根据热区类型和当前学科返回对应示例文字
- 状态栏实时显示识别进度（如：文字识别: 3/9）
- 完成后刷新右侧面板和热区列表

---

### 3.8 数据管理

#### 3.8.1 数据存储
- 使用 localStorage 持久化存储
- 存储键格式：`backstage_zones_{课本ID}_{章节ID}_p{页码}`
- 存储内容：热区数组（含位置、内容、音频状态等）

#### 3.8.2 教材数据
| 学科 | 出版社 | 课本数量 |
|------|--------|----------|
| 英语 | 沪教版 | 11本（三年级~六年级，含新版） |
| 语文 | 人教版 | 12本（一年级~六年级，含新版） |

---

### 3.9 消息提示（Toast）

- 位于页面右上角
- 支持四种类型：成功（绿色）、错误（红色）、信息（蓝色）、警告（黄色）
- 自动 3 秒后淡出消失
- 带有对应 emoji 图标

## 五、操作流程

```
选择学科 → 选择教材(出版社) → 选择课本 → 选择章节
    ↓
加载课本页面图片
    ↓
添加热区（手动绘制 或 自动识别）
    ↓
调整热区位置和大小（拖拽移动 / 拉伸手柄 / 手动输入坐标）
    ↓
识别文字（批量OCR 或 手动编辑）
    ↓
生成音频（选择音色 → 批量生成 → 试听确认）
    ↓
保存全部数据
```
